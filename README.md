# 🛍️ Интернет-магазин

Полнофункциональный интернет-магазин на чистом JavaScript и Node.js.

## 🚀 Возможности

- 📦 Каталог товаров с фильтрацией по категориям
- 🛒 Корзина покупок с слайдером
- 👤 Система пользователей (гость, пользователь, администратор)
- 💰 Система оплаты
- ⭐ Избранные товары
- 📊 Админ-панель с статистикой
- 💬 Система отзывов
- 🌙 Темная/светлая тема
- 📱 Полная адаптивность

## 🛠 Технологии

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **Backend**: Node.js (чистый, без фреймворков)
- **Хранение данных**: JSON-файлы
- **Деплой**: Render.com
